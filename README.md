## Usage 
* Clone this repository
  ```
  https://github.com/KISHOREkevin/.dotfiles.git
  ```
* Change the `install.sh` file permission to executable
  ```
  chmod +x install.sh
  ```
* Just run `install.sh`
  ```
  ./install.sh
  ```
* type `startx` and hit enter.
